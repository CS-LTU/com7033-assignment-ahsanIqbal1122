# Contributing to Stroke Prediction System

## Getting Started

Fork the repository and clone it locally. Create a new branch for your changes.

## Development Setup

1. Install Python 3.8+
2. Install dependencies: `pip install -r requirements.txt`
3. Set up the database: `python -c "from database.db import init_db; init_db()"`
4. Run the app: `python app.py`

## Code Style

- Follow PEP 8 guidelines
- Use meaningful variable names
- Add docstrings to functions and classes
- Keep functions focused and small

## Making Changes

- Write clear commit messages
- Test your changes before submitting
- Update documentation if needed
- Make sure existing tests pass

## Pull Requests

- Describe what your PR does
- Reference any related issues
- Keep PRs focused on a single feature or fix
- Be responsive to feedback

## Reporting Issues

- Check if the issue already exists
- Provide steps to reproduce
- Include error messages and logs
- Mention your environment (OS, Python version, etc.)

## Security

Report security vulnerabilities privately to the course instructor.

## Questions

For questions about the project, open an issue or contact the maintainer.
